

	Emb.Poseidon.Oracle.DataAccess
	uibasoft - http://www.uibasoft.com


  ¿Qué es?
  -----------

	Emb.Poseidon.Oracle.DataAccess Componente de Framework Poseidon es un 
	componente que ofrece clases de acceso a datos por medio de ADO.NET 
	usando el paquete Oracle.ManagedDataAccess


  Última Version
  ------------------

	Detalles de la última version puedes revisar:
	https://github.com/uibasoft/orcladonet/releases

  Documentación
  -------------

	Toda la documentación y especificacion de cada una de las clases
	del componente están en desarrollo y puedes encontrarla en:
	https://github.com/uibasoft/orcladonet/wiki


  Instalación
  ------------

	Para instalar el componente puedes usar el gestor de páquedes
	desde la interfaz grafica o la consola de administración de 
	paguetes nuget ingresando:

	Emb.Poseidon.Oracle.DataAccess
